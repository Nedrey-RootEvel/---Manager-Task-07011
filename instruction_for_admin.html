🌟 Введение

Менеджер записей - это мощная система для управления задачами, заметками, файлами и комментариями с гибкой системой прав доступа.

👑 Роль Администратора: Вы имеете полный доступ ко всем функциям системы, включая управление пользователями, вкладками и всеми записями.

Основные возможности:

  • 📑 Создание и управление тематическими вкладками
  • 📝 Работа с записями (заголовок, система, объект, описание)
  • 🎨 Цветовая маркировка записей
  • 📎 Прикрепление файлов (до 500MB)
  • 💬 Комментирование записей
  • 🔍 Гибкая система фильтров
  • 👥 Управление пользователями и правами
  • 📊 Экспорт данных в Excel и HTML

🔐 Авторизация

Вход в систему

  1. На странице входа введите ваше имя пользователя
  2. Введите пароль
  3. Нажмите кнопку "Войти"
Данные по умолчанию: admin / 1

Система запомнит ваш последний логин с помощью cookie.

Смена пароля

  1. Нажмите кнопку 🔑 рядом с вашим именем
  2. Введите текущий пароль
  3. Введите новый пароль
  4. Подтвердите новый пароль
  5. Нажмите "Сохранить"
Важно: Пароль нельзя восстановить, только сменить через администратора.

Выход из системы

Нажмите кнопку 🚪 Выйти в правом верхнем углу.

📑 Управление вкладками

Создание новой вкладки

  1. Нажмите кнопку ➕ Вкладка в верхней панели
  2. Введите название вкладки
  3. Выберите цвет вкладки из палитры (48 цветов)
  4. Настройте права доступа для вкладки:
    • ➕ Добавление записей
    • ✏️ Редактирование
    • 🗑️ Удаление
  5. Нажмите "Сохранить"
Совет: Используйте цвета для визуального разделения вкладок по проектам или отделам.

Редактирование вкладки

  1. Нажмите на иконку ⚙️ на нужной вкладке
  2. Измените название, цвет или права доступа
  3. Нажмите "Сохранить"

Администратор может редактировать любые вкладки, включая созданные другими пользователями.

Удаление вкладки

  1. Откройте настройки вкладки ⚙️
  2. Нажмите кнопку 🗑️ Удалить вкладку
  3. Подтвердите удаление
Что происходит при удалении: Вкладка перемещается в корзину вкладок, все записи сохраняются и могут быть восстановлены вместе с вкладкой.

Корзина вкладок

  1. Нажмите кнопку 🗑️ Корзина вкладок
  2. Просмотрите список удаленных вкладок
  3. Нажмите ♻️ Восстановить для восстановления вкладки
Важно: Восстановление вкладки автоматически восстанавливает все записи, которые в ней находились.

Управление порядком вкладок

  1. Нажмите кнопку 📑 Управление
  2. В открывшемся окне перетаскивайте вкладки за иконку ☰
  3. Нажмите "Сохранить порядок"

Порядок вкладок сохраняется для всех пользователей.

Управление доступом к вкладке

  1. Откройте настройки вкладки ⚙️
  2. Нажмите 👥 Просмотр пользователей - увидеть список имеющих доступ
  3. Нажмите 🔑 Управление доступом - открыть панель управления
  4. В панели управления можно:
    • Видеть пользователей с доступом
    • Отзывать доступ (кнопка "Отозвать")
    • Предоставлять доступ новым пользователям (кнопка "Предоставить")
Важно: Создатель вкладки всегда имеет к ней доступ независимо от настроек.

📝 Управление записями

Создание новой записи

  1. Выберите нужную вкладку
  2. Нажмите кнопку ➕ Добавить
  3. Заполните поля:
    • Заголовок - выберите из списка или создайте свой (✏️)
    • Система - выберите из списка или создайте свою (✏️)
    • Объект - выберите из списка или создайте свой (✏️)
    • Описание - текст записи
    • Цвет - выберите цвет из палитры
  4. Нажмите "Сохранить"
Совет: Система запоминает последние выбранные значения (заголовок, система, объект, цвет) для каждого пользователя.

Редактирование записи

  1. Нажмите кнопку ✏️ в строке записи
  2. Измените необходимые поля
  3. Нажмите "Сохранить"

Администратор может редактировать любые записи, даже чужие.

Удаление записи

  1. Нажмите кнопку 🗑️ в строке записи
  2. Подтвердите удаление

Запись перемещается в корзину (вкладка "Корзина").

Восстановление записи из корзины

  1. Перейдите во вкладку 🗑️ Корзина
  2. Найдите нужную запись
  3. Нажмите кнопку ♻️
  4. Выберите способ восстановления:
    • ОК - восстановить как есть (с оригинальной датой)
    • Отмена - восстановить как новую (с текущей датой)

Управление статусами

У каждой записи есть два статуса:

  • ✓ Выполнено - отмечает задачу как выполненную (зеленый фон 40%)
  • ⭐ Актуально - отмечает актуальность записи

Статусы можно менять с помощью чекбоксов в колонке "ДАТЫ / СТАТУС".

Цветовая маркировка

Каждая запись может иметь свой цвет из палитры 48 цветов (4 ряда по 12 цветов разной насыщенности).

  • Ряд 1: Светлые цвета
  • Ряд 2: Средние цвета (40%)
  • Ряд 3: Яркие цвета (70%)
  • Ряд 4: Насыщенные цвета (90%)

🔍 Система фильтров

Фильтры расположены под вкладками. Активные фильтры подсвечиваются и отображаются в виде значков.

👤 По автору

  • Все авторы - все записи
  • Только мои - записи, созданные текущим пользователем
  • Только чужие - записи других пользователей

📌 По заголовку

Выберите конкретный заголовок из выпадающего списка.

⚙️ По системе

Фильтрация по полю "Система".

📦 По объекту

Фильтрация по полю "Объект".

🔍 Поиск по описанию

Текстовый поиск в поле "Описание". Нажмите Enter для применения.

⚡ По статусу

  • Актуальные - только актуальные записи
  • Не актуальные - только неактуальные записи
  • Выполненные - только выполненные записи

👁️ По прочтению

  • Непрочитанные - записи, которые не просматривались
  • Прочитанные - уже просмотренные записи

📎 По файлам

  • С файлами - записи с прикрепленными файлами
  • Без файлов - записи без файлов

Сброс фильтров: Нажмите кнопку 🔄 Сброс или кликните на крестик у активного фильтра.

📎 Управление файлами

Загрузка файлов

  1. Откройте запись
  2. Нажмите кнопку 📎 в колонке "ВЛОЖЕНИЯ"
  3. Перетащите файлы в область загрузки или кликните для выбора
  4. Дождитесь завершения загрузки
Ограничения: Максимальный размер файла - 500MB. Поддерживаются все типы файлов.

Просмотр файлов

  1. Кликните на счетчик файлов 📎 N в записи
  2. В открывшемся окне вы увидите список всех файлов
  3. Для изображений доступен предпросмотр (клик для увеличения)
  4. Нажмите 📥 Скачать для загрузки файла

Удаление файлов

  1. Откройте окно с файлами записи
  2. Переключитесь на вкладку 📎 Активные
  3. Нажмите 🗑️ Удалить у нужного файла
  4. Подтвердите удаление

Файл перемещается в корзину файлов.

Восстановление файлов

  1. Откройте окно с файлами записи
  2. Переключитесь на вкладку 🗑️ Корзина
  3. Нажмите ♻️ Восстановить у нужного файла

💬 Комментарии

Добавление комментария

  1. Нажмите кнопку 💬 в строке записи
  2. Введите текст комментария
  3. Нажмите 💬 Отправить

Просмотр комментариев

Количество комментариев отображается в блоке под описанием записи. Кликните на блок для просмотра всех комментариев.

Удаление комментариев

Администратор может удалять любые комментарии. Нажмите 🗑️ рядом с комментарием.

👥 Управление пользователями

Создание нового пользователя

  1. Нажмите кнопку 👥 Пользователи
  2. Нажмите ➕ Добавить пользователя
  3. Заполните поля:
    • Полное имя - для отображения
    • Логин - для входа в систему
    • Пароль - для входа
  4. Настройте права доступа (см. ниже)
  5. Выберите доступные вкладки
  6. Нажмите "Сохранить"

Настройка прав доступа

Для обычных пользователей доступны следующие права:

  • 👁️ Просмотр записей - базовая возможность
  • ➕ Добавление записей - создание новых записей
  • ✏️ Редактирование - с опцией "только свои"
  • 🗑️ Удаление - с опцией "только свои"
  • 💬 Комментирование - с опцией "только свои"
  • ⚡ Изменение статусов - с опцией "только свои"
  • ➕ Создание вкладок - с опцией "только свои"
  • 👥 Просмотр пользователей вкладок - с опцией "только свои"
  • 🔑 Управление доступом к вкладкам - с опцией "только свои"
  • 📄 Просмотр файлов - с опцией "только свои"
  • 📎 Загрузка файлов - с опцией "только к своим записям"
  • 🗑️ Удаление файлов - с опцией "только свои"
  • 🗑️ Удаление комментариев - с опцией "только свои"

Кнопки выбора: Используйте кнопки "✅ Выбрать все" и "❌ Снять все" для быстрой настройки прав.

Доступ к вкладкам

В разделе "Доступные вкладки" отметьте вкладки, к которым пользователь будет иметь доступ. Администратор имеет доступ ко всем вкладкам автоматически.

Редактирование пользователя

  1. Нажмите 👥 Пользователи
  2. Нажмите ✏️ в строке пользователя
  3. Измените необходимые поля
  4. Нажмите "Сохранить"

Удаление пользователя

  1. Нажмите 👥 Пользователи
  2. Нажмите 🗑️ в строке пользователя
  3. Подтвердите удаление
Внимание: Пользователя "admin" нельзя удалить.

📊 Экспорт данных

Экспорт в Excel

  1. Нажмите кнопку 📊 Excel
  2. Файл будет автоматически скачан

Excel экспорт учитывает все примененные фильтры и текущую вкладку.

Экспорт в HTML

  1. Нажмите кнопку 🌐 HTML
  2. Откроется новая вкладка с отформатированной версией для печати

HTML экспорт также учитывает все примененные фильтры.
