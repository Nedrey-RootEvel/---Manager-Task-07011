# ---Manager-Task-07011
Система записей задач авторизация настройка и сервер все в одном файле скрипте
Основные возможности:
Создание записей с заголовком, описанием, цветом, системой и объектом
Управление пользователями с гибкими правами доступа (администраторы и обычные пользователи)
Работа с несколькими вкладками (категориями записей)
Загрузка и просмотр файлов (до 500 МБ)
Комментирование записей

Организация работы:
Фильтрация записей по автору, заголовку, системе, объекту, статусу и наличию файлов
Отметка о прочтении записей и файлов
Возможность отмечать задачи как "выполнено" или "актуально"
Корзина для удаленных записей и вкладок
Администрирование:
Создание/редактирование/удаление пользователей с настройкой прав
Управление доступом к вкладкам
Экспорт данных в Excel и HTML
Приложение работает на PHP с файловым хранением данных (JSON) и подходит для небольшой команды.
