🌟 Введение

Менеджер записей - это система для ведения задач, заметок и документов с удобной системой фильтрации и совместной работы.

👤 Ваша роль: Вы обычный пользователь. Ваши возможности определяются правами, которые выдал администратор.

Ваши возможности:

  • 📝 Создание и управление записями
  • 🎨 Цветовая маркировка записей
  • 📎 Прикрепление файлов к записям
  • 💬 Комментирование
  • 🔍 Использование фильтров
  • 📊 Экспорт данных
  • 📑 Создание своих вкладок (если есть право)

🔐 Авторизация

Вход в систему

  1. На странице входа введите ваш логин (выдан администратором)
  2. Введите ваш пароль
  3. Нажмите кнопку "Войти"

Система запомнит ваш логин для следующего входа.

Смена пароля

  1. Нажмите кнопку 🔑 рядом с вашим именем
  2. Введите текущий пароль
  3. Введите новый пароль
  4. Подтвердите новый пароль
  5. Нажмите "Сохранить"
Важно: Пароль нельзя восстановить. Если забудете - обратитесь к администратору.

Выход из системы

Нажмите кнопку 🚪 Выйти в правом верхнем углу.

📑 Работа с вкладками

Доступные вкладки

В верхней части страницы вы видите вкладки, к которым у вас есть доступ:

  • Все записи - все записи из доступных вам вкладок
  • Пользовательские вкладки - созданные вами или другими пользователями
  • Корзина - удаленные вами записи (или все, если вы администратор)

Цифра на вкладке показывает количество непрочитанных записей.

Создание новой вкладки

Доступно, если у вас есть право "Создание вкладок".

  1. Нажмите кнопку ➕ Вкладка
  2. Введите название вкладки
  3. Выберите цвет из палитры
  4. Настройте права доступа для других пользователей
  5. Нажмите "Сохранить"
Важно: Создатель вкладки всегда имеет к ней полный доступ.

Редактирование своих вкладок

Вы можете редактировать только те вкладки, которые создали сами.

  1. Нажмите на иконку ⚙️ на своей вкладке
  2. Измените название, цвет или права доступа
  3. Нажмите "Сохранить"

📝 Управление записями

Создание новой записи

  1. Выберите вкладку, в которую хотите добавить запись
  2. Нажмите кнопку ➕ Добавить
  3. Заполните поля (система и объект могут быть пустыми):
Поля ввода с автозаполнением:
  • Заголовок - можно выбрать из списка или создать свой (выберите "✏️ Свой заголовок")
  • Система - можно выбрать существующую или создать свою
  • Объект - можно выбрать существующий или создать свой
  • Описание - основной текст записи
  • Цвет - выберите цвет из палитры

Как удалить созданный вами элемент: Выберите его в списке и нажмите кнопку 🗑️ справа. Стандартные элементы (Задача, Сообщение и т.д.) удалить нельзя.

Прилипание значений: Система запоминает последние выбранные вами заголовок, систему, объект и цвет. При следующем создании записи они будут подставлены автоматически.

Редактирование записи

Вы можете редактировать:

  • Свои записи - всегда (если есть право на редактирование)
  • Чужие записи - только если администратор дал право редактировать все записи
  1. Нажмите кнопку ✏️ в строке записи
  2. Внесите изменения
  3. Нажмите "Сохранить"

Удаление записи

Вы можете удалять:

  • Свои записи - всегда (если есть право на удаление)
  • Чужие записи - только если есть право удалять все записи
  1. Нажмите кнопку 🗑️ в строке записи
  2. Подтвердите удаление

Запись перемещается во вкладку "Корзина".

Восстановление записи из корзины

Вы можете восстановить только свои удаленные записи (администратор может любые).

  1. Перейдите во вкладку 🗑️ Корзина
  2. Найдите свою удаленную запись
  3. Нажмите кнопку ♻️
  4. Выберите способ:
    • ОК - восстановить как есть (с оригинальной датой)
    • Отмена - восстановить как новую (с текущей датой)

Управление статусами

У каждой записи есть два чекбокса в колонке "ДАТЫ / СТАТУС":

  • ✓ Выполнено - отметьте, когда задача выполнена (запись становится зеленой)
  • ⭐ Актуально - показывает, актуальна ли запись сейчас

Изменять статусы можно только у записей, к которым у вас есть доступ.

Цветовая маркировка

При создании записи вы можете выбрать один из 48 цветов (4 ряда по 12 цветов разной насыщенности).

  • Ряд 1: Светлые пастельные тона
  • Ряд 2: Средние цвета (40% насыщенности)
  • Ряд 3: Яркие цвета (70% насыщенности)
  • Ряд 4: Насыщенные цвета (90% насыщенности)

🔍 Система фильтров

Фильтры расположены под вкладками. Они помогают быстро найти нужные записи.

👤 По автору

  • Все авторы - показывать все записи
  • Только мои - только ваши записи
  • Только чужие - только записи других пользователей

📌 По заголовку

Выберите конкретный заголовок, чтобы увидеть только записи с ним.

⚙️ По системе

Отфильтруйте записи по полю "Система".

📦 По объекту

Отфильтруйте записи по полю "Объект".

🔍 Поиск по описанию

Введите текст для поиска в описании записей. Нажмите Enter для применения.

⚡ По статусу

  • Актуальные - только актуальные записи
  • Не актуальные - только неактуальные
  • Выполненные - только выполненные

👁️ По прочтению

  • Непрочитанные - записи, которые вы еще не открывали
  • Прочитанные - уже просмотренные вами

📎 По файлам

  • С файлами - записи с прикрепленными файлами
  • Без файлов - записи без файлов

Активные фильтры отображаются в виде цветных значков. Нажмите крестик на значке, чтобы убрать фильтр. Кнопка 🔄 Сброс убирает все фильтры сразу.

📎 Управление файлами

Загрузка файлов

Вы можете прикреплять файлы только к своим записям (или к любым, если есть право).

  1. Откройте запись (нажмите на зеленый индикатор файлов или кнопку 📎)
  2. Перетащите файлы в область загрузки или кликните для выбора
  3. Дождитесь завершения загрузки
Ограничения: Максимальный размер файла - 500MB. Можно загружать любые типы файлов.

Просмотр файлов

  1. Кликните на счетчик файлов 📎 N в записи
  2. В окне вы увидите список файлов с информацией:
    • Имя файла
    • Дата загрузки
    • Кто загрузил
    • Размер
  3. Для изображений доступен предпросмотр (кликните на картинку для увеличения)
  4. Нажмите 📥 Скачать для загрузки файла

Удаление файлов

Вы можете удалять только свои файлы (администратор может любые).

  1. Откройте окно с файлами записи
  2. На вкладке 📎 Активные найдите нужный файл
  3. Нажмите 🗑️ Удалить
  4. Подтвердите удаление

💬 Комментарии

Добавление комментария

  1. Нажмите кнопку 💬 в строке записи
  2. Напишите текст комментария
  3. Нажмите 💬 Отправить

Вы можете комментировать записи, к которым у вас есть доступ.

Просмотр комментариев

Количество комментариев отображается под описанием записи. Кликните на этот блок, чтобы открыть окно со всеми комментариями.

Удаление комментариев

Вы можете удалять только свои комментарии (администратор может любые).

  1. Откройте окно с комментариями
  2. Найдите свой комментарий
  3. Нажмите 🗑️ рядом с ним
  4. Подтвердите удаление

👤 Управление профилем

Смена пароля

  1. Нажмите кнопку 🔑 рядом с вашим именем в верхней панели
  2. Введите текущий пароль
  3. Введите новый пароль (не менее 1 символа)
  4. Подтвердите новый пароль
  5. Нажмите "Сохранить"
Важно: Записывайте новый пароль. При утере обращайтесь к администратору.

📊 Экспорт данных

Экспорт в Excel

  1. Нажмите кнопку 📊 Excel
  2. Файл будет автоматически скачан

В экспорт попадут только записи, соответствующие текущим фильтрам.

Экспорт в HTML (для печати)

  1. Нажмите кнопку 🌐 HTML
  2. Откроется новая вкладка с красиво оформленным списком записей
  3. Используйте Ctrl+P для печати

HTML экспорт также учитывает все примененные фильтры.
